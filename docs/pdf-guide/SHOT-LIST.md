## Screenshot kaise le (Windows)

- **`Win + Shift + S`** dabao → area select karo → phir Paint mein paste karke PNG save karo, **ya**
- Browser ka full-page screenshot extension use karo.
- **Tip:** Browser window ko thoda chauda rakho (taaki left sidebar bhi dikhe). Har shot mein poora content area aa jaye.
- Format: **PNG** (ya JPG chalega). File ka naam **bilkul** neeche jaisa rakho.
